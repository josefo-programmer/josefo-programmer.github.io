Sobre Mí

Soy un apasionado estudiante de programación con 2 años de experiencia, especializado en el ecosistema de Java. Además de Java, tengo conocimientos en Python y estoy ampliando mi perfil hacia la analítica de datos, utilizando herramientas como Postman para la prueba de APIs, y gestionando bases de datos con phpMyAdmin. Mi objetivo es combinar el desarrollo de software robusto con el análisis de datos para crear soluciones completas e inteligentes.
